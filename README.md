// TENTANG APLIKASI DOMATIO CAFE //

Aplikasi Domatio Cafe ini adalah aplikasi yang khusus dirancang untuk admin untuk mengelola
system pemesanan yang ada pada domatio cafe. Aplikasi ini bertujuan untuk memudahkan
system pengelolaan pemesanan agar lebih efisien dan modern dengan system aplikasi yang dijalankan
melalui perangkat computer. Manfaat dari system pemesanan ini adalah proses input yang dilakukan
oleh kasir menjadi lebih mudah tanpa harus mencatat pesanan dari kostumer secara manual.

============================================================================================================

// FITUR //

# Tampilan antarmuka yang mudah dipahami sehingga bisa digunakan dengan mudah dan cepat
# Mencetak struk pemesanan dengan mudah
# Perhitungan potongan pajak otomatis pada pencetakan struk pemesanan
# Pemilihan meja makan beserta nomor meja dan jumlah kursi

============================================================================================================

// CARA PENGGUNAAN //

- Sign in akun baru admin (sebagai verifikasi admin domatio cafe)
- Login dengan akun yang sudah dibuat
- Pilih menu "Pesan" untuk proses pemesanan makanan, dan menu "Exit" untuk keluar dari aplikasi
- Pilih meja yang kosong beserta jumlah kursinya
- Pilih jumlah menu yang akan dipesan, lalu tekan "Beli" untuk mengonfirmasi pembelian
- Setelah semua menu sudah dipesan, tekan "Total" untuk menjumlahkan total biaya pemesanan
- Tekan "Cetak" untuk mencetak struk pemesanan
